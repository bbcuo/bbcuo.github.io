# 📚 图书管理系统

## 📖 项目简介
基于 Java 控制台的图书管理系统，支持图书的增删改查和借阅归还功能。
跟着老师一步步完成的实战项目，巩固面向对象、集合、IO流等知识。

## 🛠️ 技术栈
- Java
- 面向对象编程（封装、继承、多态）
- ArrayList 集合
- 文件 IO 流

## ✨ 功能列表
- ✅ 添加图书
- ✅ 删除图书
- ✅ 修改图书信息
- ✅ 查询图书（按书名/作者）
- ✅ 显示全部图书
- ✅ 借阅图书
- ✅ 归还图书

## 📁 项目结构
src/
├── Book.java           # 图书实体类
├── BookManager.java    # 图书管理核心逻辑
├── LibraryApp.java     # 主程序入口（菜单交互）
└── books.txt           # 数据持久化文件

## 🚀 运行方式
1. 用 IntelliJ IDEA 打开项目
2. 运行 `LibraryApp.java` 的 main 方法
3. 根据控制台菜单提示操作

## 📝 学习心得
通过这个项目，我深入理解了：
- 类的封装：将图书属性和行为封装在 Book 类中
- 集合的使用：用 ArrayList 存储和管理图书数据
- IO 操作：通过文件读写实现数据持久化
- 面向对象设计：将界面、业务逻辑、数据分离
